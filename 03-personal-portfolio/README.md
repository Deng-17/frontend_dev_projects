# Personal Portfolio
This is a CSS-styled version of the previous basic HTML website, with responsive
layouts, color and typography.

## Overview
**Goal**: Style the previous basic HTML-only website using various CSS techniques
to enhance appearance and responsiveness.

**Must include**:
- [x] A fully styled, responsive website with the same structure of the previous project.
- [x] Consistent use of a chosen color scheme and typography.
- [x] Proper use of CSS techniques like Flexbox, media queries, and the box model.
- [x] A responsive navigation bar and well-styled contact form.
